# 📊 Monitoramento Estratégico: Rede Municipal de Recife (Censo 2025)

Este repositório apresenta uma solução de inteligência de dados voltada para o suporte à gestão pública, utilizando a **Rede Municipal de Recife** como estudo de caso para o monitoramento de indicadores educacionais complexos.

## 🎯 Objetivo do Projeto
Otimizar a tomada de decisão da equipe de parcerias do Instituto, transformando microdados brutos do INEP em indicadores acionáveis, como o proposto **IAP (Índice de Adesão e Percepção)**.

## 🛠️ Stack Tecnológica
- **Engenharia de Dados:** R (Linguagem R) para processamento de Big Data.
- **Tratamento:** Bibliotecas `dplyr` e `readr` (otimização de memória via *Column Projection*).
- **Extração & Análise:** SQL (Google BigQuery).
- **Visualização:** Looker Studio.
- **Documentação:** GitHub.

## 🚀 Diferenciais Técnicos e Senioridade
- **Dados em Tempo Real (Censo 2025):** Processamento da base mais recente do INEP, garantindo que a análise reflita o cenário atual da rede municipal.
- **Eficiência Computacional:** Uso de scripts R otimizados para lidar com datasets de +150MB em ambiente de nuvem com restrição de RAM.
- **Visão de Negócio:** Foco em identificar gargalos na "Rede B" (boa performance, mas desafios técnicos de infraestrutura).

## 📁 Organização do Repositório
- `/datasets`: Bases consolidadas em CSV (Censo 2025 e IDEB Recife).
- `/queries`: Scripts R e SQL para tratamento e limpeza dos dados.
