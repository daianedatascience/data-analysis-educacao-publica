# Análise de Dados - Educação Pública

Este projeto tem como objetivo analisar dados educacionais da rede pública com foco
em geração de indicadores para apoio à tomada de decisão.

## Ferramentas utilizadas
- SQL
- Google BigQuery
- Looker Studio
- Google Forms
- GitHub

## Etapas do projeto
1. Coleta de dados educacionais (INEP, Censo Escolar, IDEB)
2. Armazenamento e tratamento dos dados no BigQuery
3. Análises exploratórias com SQL
4. Criação de indicadores educacionais
5. Construção de dashboard no Looker Studio
6. Documentação das análises

## Exemplos de indicadores analisados
- Número de escolas
- Número de matrículas
- Relação aluno/professor
- Infraestrutura das escolas
- IDEB por escola
- Evolução de matrículas

## Objetivo
Demonstrar como dados educacionais podem ser utilizados para gerar insights e apoiar
gestores educacionais na tomada de decisão.
